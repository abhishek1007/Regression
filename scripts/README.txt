The code is divided into three python scripts.

To run Q1 to Q8:
	python Q1,2,3,4,5,6,7,8.py

To run Q9:
	python Q9.py

To run Q10:
	python Q10.py

Q1,2,3,4,5,7,8.py will generate all the relevant data in following format
Lambda \t Fraction \t Training Error \t Testing Error

Q9.py will plot the relevant graphs and save them in graphs named folder.

Q10.py will plot the relevant graphs and save them in graphs named folder.

To Plot graphs from output of Q1,2,3,4,5,6,7,8.py run python report_maker.py which will save the relevant graphs in folder named graphs
